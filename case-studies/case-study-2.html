A/B Testing

Context/Problem

We aimed to determine which website layout would lead to higher user engagement.

Research

We set up A/B tests to compare two different layouts and analyzed user engagement metrics, focusing on making data-driven design decisions.

Process

Based on the initial results, we made iterative changes to the layouts and tested them again, always prioritizing simplicity and user-friendliness.

Solution

The winning layout featured a simplified navigation menu and increased engagement by 20%.

Learnings

I learned how to effectively use A/B testing to make data-driven design decisions and improve user engagement.

Role

I designed the layouts and managed the A/B testing process, focusing on simplicity and accessibility.
